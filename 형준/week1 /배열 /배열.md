# 배열

- ## 1. 배열의 정의와 성질

    배열 : 메모리 상의 원소를 연속하게 배치한 자료구조

    성질

      1. O(1) 에 k번째 원소를 확인, 변경 가능

      2. 추가적으로 소모되는 메모리 양이 거의 없음

      3. Cache hit rate가 높다

      4. 메모리 상에 연속한 구간을 잡아야 해서 할당에 제약이 걸림

    기능

      1. 임의의 위치 확인, 변경 - O(1)

      2. 맨 뒤에 원소 추가 - O(1)

      3. 임의의 위치에 원소 추가, 삭제 - O(N)
